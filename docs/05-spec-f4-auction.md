# 05 · F4 大佬时间拍卖 Spec

> 变现 + 传播引爆点。节奏决议（01 §六）：**M2 运营专场（表单级，零竞价代码）→ M3 功能化**。
> M2 的 10 场真实拍卖是 M3 竞价系统的需求验证——规则先在人肉流程里跑通，再写代码。

## 1. 拍的是「场景」不是时间

| 场景示例 | 说明 |
|---|---|
| 陪开一次评审会 | 旁听 + 会后 30 分钟复盘 |
| 模拟面试 | 目标岗位级别的真实压力面 |
| 晨跑局 | 一小时非正式请教 |

- 场景由嘉宾自选 + 运营把关（避免出现「代写简历」式廉价场景拉低格调）。
- 嘉宾门槛：trustLevel ≥ 2 + 运营邀约制（M3 功能化后开放自荐，仍需审核）。
- **嘉宾预期管理**（01 §三的利益冲突原则）：邀约话术明确告知——其所在公司/部门的评价内容不因参与拍卖获得任何特殊处理。

## 2. 竞价规则：诚意盲拍 + 心动权

1. **系统定指导价**：按嘉宾段位 + 场景时长给出区间（v1 人工定价，M3 后用历史成交回归）。
2. **盲拍出价**：竞拍期 72 小时，出价人提交 `出价金额 + 「为什么是我」（200 字）`；盲拍——互相看不到出价，只看到参与人数（保留紧张感，避免狙击与串价）。
3. **心动权**：截拍后嘉宾可**绕过最高价**从全部出价中选一人——「为什么是我」的权重由嘉宾自定。保留阶层流动的浪漫，也是媒体传播点。
4. 嘉宾不行使心动权 → 最高价成交；同价先出者得。
5. **成交价公开**，形成行情榜（全站最硬鄙视链）；出价人身份默认匿名（段位可见），成交者可选是否公开。

### 行情榜

- 按嘉宾维度展示历史成交价曲线；M3 上线。
- 行情榜入口低调（克制原则），但单场成交可生成分享卡（同 03 §3.3 的 og 体系）。

## 3. M2 运营专场（表单级实现）

- 静态专场页 `app/auction/page.tsx`：嘉宾介绍 + 报名表单（出价 + 为什么是我）。
- 表单落库 `auction_bids` 简表（或外部表单工具，二选一以快为准）。
- 撮合、通知、收款、履约全人肉；**首季收入全额捐赠** + 捐赠收据公示页。
- 全捐的合规意义：平台不分润 → 无抽佣资金结算问题，M2 不碰支付通道（见 §5）。
- 验收：首季 10 场，场均竞拍 >20 人，≥1 次破圈传播（媒体自发报道或单平台 10w+ 阅读）。

## 4. M3 功能化（数据模型预案）

```
auctions
  id, host_user_id (FK), scenario_title, scenario_desc, guide_price_min/max,
  starts_at, ends_at, status (draft/live/closed/settled/cancelled),
  final_price, winner_bid_id, charity_flag, created_at
auction_bids
  id, auction_id (FK), bidder_user_id (FK), amount, reason_text (200字),
  status (active/withdrawn/won/lost), is_heart_pick (boolean), created_at
```

- 状态机：`draft → live → closed →（嘉宾 72h 内选标或默认最高价）→ settled`。
- 履约确认：双方确认完成后才结算；纠纷走人工仲裁（M3 先人工，不做自动仲裁）。

## 5. 资金与合规（M3 前必须闭环的前置项）

| 事项 | 方案 |
|---|---|
| 收款通道 | 接持牌支付机构分账产品（微信/支付宝服务商分账或银联），**平台不碰资金池** |
| 抽佣 | 10-15%，通过分账实现；公益场 0 抽佣 |
| 嘉宾个税 | 劳务报酬代扣代缴——接灵工平台或财税服务商，M3 选型 |
| 增值电信资质 | 上线变现前完成 ICP 经营性备案评估 |
| 公益捐赠 | 与具公开募捐资格的基金会合作，捐赠路径与收据公示 |

> 这张表是 M3 排期的硬前置：任何一项没闭环，拍卖只能继续以全捐运营专场形态存在。

## 6. 验收标准

| 阶段 | 指标 | 目标 |
|---|---|---|
| M2 运营季 | 场次 / 场均竞拍 / 破圈 | 10 场 / >20 人 / ≥1 次 |
| M3 功能化 | 心动权行使率 | 留观：过高说明价格机制失效，过低说明叙事没立住 |
| M3 功能化 | 履约完成率 | >95%，纠纷率 <2% |
| M3 功能化 | 平台佣金收入 | 首个变现场景，目标值由 M2 实际成交价定 |
